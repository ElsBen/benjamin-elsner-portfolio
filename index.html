Frontend:

HTML (Hypertext Markup Language)

HTML ist die grundlegende Sprache für das Erstellen von Webseiten. Es legt das Gerüst fest und ermöglicht es, verschiedene Elemente wie Text, Bilder, Links und mehr zu strukturieren. Mit HTML wird die Basis für das Layout und die Inhalte einer Webseite geschaffen. Diese Technologie verwende ich in all meinen Projekten und sie ist für meine Arbeit unverzichtbar. Dank Emmet, das mir mit Kürzeln ganze HTML-Strukturen inklusive Attribute und verschachtelte Elemente auf einmal erstellt, wird das Arbeiten mit HTML deutlich effizienter und schneller.

CSS (Cascading Style Sheets)

CSS ist essenziell für eine ansprechende Gestaltung von Webseiten. Es definiert Einrückungen, Abstände, Farben, Größen und Positionen. Mit CSS können Variablen erstellt, Animationen und Interaktionen gesteuert werden, wodurch es sich in seiner Funktionalität einer Programmiersprache annähert. CSS ist mein wichtigstes Instrument, um eine hervorragende User Experience zu schaffen. Meine erste Methode, um ein responsives Design zu realisieren, war das Grid-System. Danach habe ich mich intensiv mit Flexbox beschäftigt, um noch flexiblere und dynamischere Layouts zu entwickeln. Zurzeit beschäftige ich mich mit Animationen und dynamischen Effekten in CSS, um die Benutzererfahrung weiter zu verbessern.

JavaScript

JavaScript ist eine dynamische Programmiersprache, die vor allem für die Entwicklung interaktiver Elemente auf Webseiten verwendet wird. Sie ermöglicht die Manipulation von HTML und CSS, um benutzerfreundliche und ansprechende Anwendungen zu erstellen. Darüber hinaus wird JavaScript zunehmend auch für serverseitige Programmierung und die Entwicklung komplexer Anwendungen eingesetzt, was seine Vielseitigkeit und Bedeutung in der modernen Webentwicklung unterstreicht. Mit dieser Technologie habe ich Formulardaten, Validierungen und Änderungen sowie Animationen, dynamische Elemente und das Speichern von Usereingaben erfolgreich umgesetzt. Dabei setze ich auf objektorientierte Programmierung und strukturiere meinen Code bevorzugt mit Modulen, um eine klare und wartbare Architektur zu gewährleisten.

TypeScript

TypeScript ist eine von Microsoft entwickelte Programmiersprache, die eine statische Typisierung für JavaScript hinzufügt. Sie ermöglicht Entwicklern, Fehler frühzeitig zu erkennen und den Code besser zu strukturieren, was die Wartbarkeit erhöht. TypeScript wird häufig in größeren Projekten eingesetzt, da es eine klare Typenbeschreibung und moderne Features bietet, die die Entwicklung komplexer Anwendungen erleichtern. Diese Erweiterung von JavaScript war auf dem Weg zum Framework Angular ein wichtiger Baustein und hat mir geholfen, ein tieferes Verständnis für die Programmierung zu entwickeln. Durch die Typisierung, abstrakte Klassen, Interfaces und weitere fortgeschrittene Features konnte ich einen sicheren Umgang mit Daten und Datentypen erlernen.

Frameworks:

Angular

Angular ist ein plattformübergreifendes Framework für die Entwicklung von dynamischen Webanwendungen, das von Google unterstützt wird. Es nutzt TypeScript, um eine modulare und komponentenbasierte Architektur zu ermöglichen, wodurch die Wartung und Erweiterung von Anwendungen erleichtert wird. Mit Angular können Entwickler komplexe Benutzeroberflächen erstellen, die reaktiv und effizient sind, und es bietet zahlreiche integrierte Funktionen wie Routing, Formularverwaltung und HTTP-Client für die Entwicklung moderner Webanwendungen.

Backend:

Node.JS

Node.js ist eine serverseitige JavaScript-Laufzeitumgebung, die es ermöglicht, JavaScript außerhalb des Browsers auszuführen. Es basiert auf der V8-JavaScript-Engine von Google und bietet ein eventbasiertes, nicht blockierendes I/O-Modell, das eine hohe Leistungsfähigkeit und Effizienz bei der Entwicklung skalierbarer Netzwerk- und Webanwendungen ermöglicht. Mit einer Vielzahl von verfügbaren Paketen über npm (Node Package Manager) können Entwickler schnell funktionale Serveranwendungen und APIs erstellen.

Versionierung:

Git, GitHub

Git ist ein verteiltes Versionskontrollsystem, das Entwicklern ermöglicht, Änderungen an Codeprojekten effizient zu verfolgen und zu verwalten. Es ermöglicht das Arbeiten an mehreren Versionen eines Projekts gleichzeitig und erleichtert die Zusammenarbeit im Team. GitHub ist eine Plattform, die auf Git aufbaut und zusätzliche Funktionen wie Issue-Tracking, Code-Review und Projektmanagement bietet, was die Zusammenarbeit und Veröffentlichung von Softwareprojekten weiter vereinfacht.

Werkzeuge:

Visual Studio Code

Visual Studio Code (VS Code) ist ein leichtgewichtiges, quelloffenes Code-Editor-Tool, das von Microsoft entwickelt wurde. Es bietet eine leistungsstarke und flexible Entwicklungsumgebung mit Unterstützung für zahlreiche Programmiersprachen wie JavaScript, Python, C++, und viele mehr. VS Code zeichnet sich durch seine intuitive Benutzeroberfläche, integrierte Git-Unterstützung, Debugging-Tools und eine Vielzahl von Erweiterungen aus, die es Entwicklern ermöglichen, ihre Arbeitsabläufe effizient anzupassen. Durch die breite Integration von Plugins und Anpassungsmöglichkeiten wird VS Code zu einer bevorzugten Wahl sowohl für Einsteiger als auch für professionelle Entwickler, um Projekte schneller und übersichtlicher zu realisieren.

Node Package Manager

NPM (Node Package Manager) ist der standardmäßige Paketmanager für Node.js und eine zentrale Plattform zur Verwaltung von JavaScript-Bibliotheken und -Abhängigkeiten. Es ermöglicht Entwicklern, externe Pakete einfach in ihre Projekte zu integrieren, Abhängigkeiten zu installieren und diese über ein zentrales Repository zu verwalten. NPM bietet eine riesige Sammlung von Open-Source-Bibliotheken und Tools, die die Entwicklung von Anwendungen erleichtern und beschleunigen. Mit Befehlen wie "npm install", "npm run" und "npm update" können Entwickler ihre Projekte effizient pflegen, testen und veröffentlichen. NPM erleichtert zudem die gemeinsame Nutzung von Paketen und vereinfacht das Deployment durch eine strukturierte und automatisierte Handhabung von Abhängigkeiten.

XAMPP

XAMPP ist eine kostenlose, plattformübergreifende Software, die eine vollständige Entwicklungsumgebung für Webanwendungen bereitstellt. Es enthält wichtige Komponenten wie Apache (Webserver), MySQL (Datenbank), PHP und Perl, die es Entwicklern ermöglichen, lokal Webseiten und Anwendungen zu erstellen und zu testen, bevor sie online veröffentlicht werden. XAMPP ist einfach zu installieren und bietet eine benutzerfreundliche Oberfläche, über die man die verschiedenen Dienste wie den Webserver oder die Datenbank mit nur wenigen Klicks starten oder stoppen kann. Es eignet sich ideal für die lokale Entwicklung von PHP-basierten Projekten und erleichtert das Testen und Debuggen von Webanwendungen in einer realistischen Umgebung.

Bootstrap

Bootstrap ist ein beliebtes, quelloffenes Frontend-Framework zur Entwicklung von responsiven und mobilen Webseiten. Es bietet eine Sammlung vorgefertigter HTML-, CSS- und JavaScript-Komponenten wie Navigationsleisten, Buttons, Formulare und Grid-Systeme, die es Entwicklern ermöglichen, moderne und ansprechende Benutzeroberflächen schnell zu erstellen. Bootstrap zeichnet sich durch seine Flexibilität und Anpassungsfähigkeit aus, was das Design für verschiedene Bildschirmgrößen und Geräte erleichtert. Durch die Nutzung von Bootstrap können Entwickler viel Zeit sparen, da sie auf bewährte, plattformübergreifende Bausteine zurückgreifen können, um professionell aussehende Webseiten zu erstellen, ohne von Grund auf eigene CSS- und JavaScript-Lösungen schreiben zu müssen.

Projekt: Responsive Webseite mit Leistungsboxen, Bildergalerie und Kontaktformular

Beschreibung:

Diese voll funktionsfähige Webseite wurde von mir mit HTML, CSS und JavaScript (Module) entwickelt. Sie bietet ein responsives Design und enthält mehrere interaktive Elemente.

  • Features:
    • Leistungsboxen und Kontaktformular: Nutzer können Leistungsboxen auswählen, die im Kontaktformular übernommen und zur Überprüfung vor der Speicherung angezeigt werden.
    • Bildergalerie mit Großansicht: Bilder werden über JavaScript dynamisch geladen. Die Dateinamen werden in einem Array gespeichert, und eine Schleife fügt die Bilder mithilfe einer Variablen für den Dateipfad ein. Die Großansicht wird mit JavaScript und CSS realisiert.
    • Optimierte Bilddarstellung: Alle Bilder sind im .avif-Format, haben eine Mindestgröße für die Layout-Stabilität und sind mit loading="lazy" versehen, um nur bei Bedarf im Viewport geladen zu werden.
  • Formular-Logik:
    • Datenspeicherung und Validierung: Das Kontaktformular verwendet localStorage, um Eingaben zwischenzuspeichern. Vor der endgültigen Speicherung werden die Daten in einem Bestätigungsfenster angezeigt. Nutzer können ihre Eingaben anpassen, ohne dass das Formular zurückgesetzt wird.
  • Zukunftspläne: Ein Backend mit Node.js ist zur Weiterentwicklung der Seite geplant.

Projekt: Finanzapplikation zur Verwaltung von Einnahmen und Ausgaben

Beschreibung:

Diese Finanzapplikation wurde in einem Onlinekurs mit HTML, CSS und JavaScript erstellt. Zur besseren Strukturierung wurden JavaScript-Module und Klassen genutzt.

  • Funktionalitäten:
    • Einnahmen und Ausgaben: Die Applikation nimmt Einnahmen, Ausgaben und das zugehörige Datum entgegen und bilanziert diese sowohl monatlich als auch jährlich.

    • Interaktive Buttons: Einfache Eingabe von Einnahmen und Ausgaben über dedizierte Buttons.
  • Technologien: HTML, CSS und JavaScript (mit Modulen und Klassen für eine saubere Strukturierung).
